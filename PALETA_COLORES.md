## 🎬 GUÍA PARA COLOR GRADING EN EDITOR

### Día 1: NEGRO A BLANCO
```
Inicio:   Negro puro (#000000)
Transición: Desaturación completa
Final:    Blanco puro (#FFFFFF)
Contraste: Máximo
Saturación: 0% al inicio, aumentar ligeramente al final
```

### Día 2: AZUL CÓSMICO
```
Tono: Azul profundo (220°-240°)
Saturación: 70-80%
Luminancia: 40-50%
Contraste: Moderado
Curvas: Sombras azules, highlights claros
```

### Día 3: VERDE VIBRANTE
```
Inicio:   Marrón tierra (#8B4513)
Transición: Gradual a verde
Final:    Verde vibrante (#22A839)
Saturación: Aumentar significativamente en time-lapse
Brillo: Aumentar hacia el final
```

### Día 4: DORADO ÉPICO ⭐
```
Dominante: Dorado/Amarillo (#FFD700)
Saturación: Máxima (100%)
Luminancia: Muy alta (70-80%)
Contraste: Alto para dramático
Efectos: Bloom, glow, lens flares
```

### Día 5: AZUL TROPICAL
```
Fase Marina: Azul tropical (#00BFFF), saturación alta
Fase Aérea: Cielo azul (#87CEEB), más claro
Transición: Gradual, no abrupta
Saturación: 75-85% en colores de aves
```

### Día 6A: DORADO CÁLIDO
```
Tono: Dorado/Naranja (#FFB347)
Temperatura: Muy cálida (K 5500+)
Saturación: 60-70%
Contraste: Moderado-bajo (natural)
Brillo: Reducido (~70% para atmósfera sepia)
```

### Día 6B: DORADO SERENO ❤️
```
Tono: Dorado muy suave (#FFE5B4)
Temperatura: Cálida pero serena
Saturación: Baja (~40-50%)
Contraste: Bajo (suave, contemplativo)
Brillo: Natural, no excesivamente brillante
Efecto especial: Desaturación parcial (90% en colores)
```
