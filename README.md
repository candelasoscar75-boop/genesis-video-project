# 🎬 PROYECTO: "Dios Ordena el Universo"
## Video Cinematográfico Basado en Génesis 1:1-27

**Duración:** 3 minutos 30 segundos (3:30)  
**Formato:** 16:9 (Horizontal para YouTube)  
**Idioma:** Español Latino  
**Estilo:** Cinematográfico, Épico, Hiperrealista, Majestuoso  

---

## 📋 ÍNDICE DE CONTENIDOS

| Archivo | Descripción |
|---------|-------------|
| **GUION_COMPLETO.md** | Narración exacta con timecodes sincronizados |
| **STORYBOARD.md** | Referencias visuales detalladas por escena |
| **PALETA_COLORES.md** | Códigos HEX exactos para color grading |
| **scripts/** | Herramientas Python automatizadas |
| **subtitulos/genesis_video_es.srt** | Subtítulos listos para importar |
| **audio/especificaciones_audio.json** | Especificaciones de audio completas |

---

## 🎯 VISIÓN GENERAL

Este proyecto narra la **Creación del Universo** según Génesis 1:1-27 en **6 días épicos**:

1. **DÍA 1 (0:00-0:30):** Luz y Tinieblas
2. **DÍA 2 (0:30-0:55):** Firmamento
3. **DÍA 3 (0:55-1:30):** Tierra y Vegetación
4. **DÍA 4 (1:30-2:00):** Sol, Luna y Estrellas ⭐ **CLÍMAX 1**
5. **DÍA 5 (2:00-2:35):** Criaturas Marinas y Aves
6. **DÍA 6 (2:35-3:30):** Animales y Humanidad ❤️ **CLÍMAX 2**

---

## 🚀 CÓMO USAR ESTE REPOSITORIO

**Para Editores de Video:**
```bash
1. Descarga GUION_COMPLETO.md
2. Lee STORYBOARD.md para referencias visuales
3. Importa subtitulos/genesis_video_es.srt
```

**Para Compositores/Diseñadores de Sonido:**
```bash
1. Abre audio/especificaciones_audio.json
2. Consulta timecodes exactos de clímax
```

**Para Artistas VFX:**
```bash
1. Lee STORYBOARD.md completamente
2. Usa PALETA_COLORES.md para color grading
```
